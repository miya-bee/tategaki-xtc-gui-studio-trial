# TategakiXTC GUI Studio Trial

縦書きXTC GUI Studio の Windows向け体験版配布ページです。

この体験版では、xteink X4 / X3 などで利用できる XTC / XTCH 形式への変換を試すことができます。

## 体験版の制限

* 出力は先頭10ページまでです。
* Pythonのインストールは不要です。
* Windows向けのポータブル版です。
* zipを展開してから起動してください。
* `.exe` だけを取り出して移動しないでください。

## ダウンロード

最新版の体験版は、Releases からダウンロードできます。

https://github.com/miya-bee/tategaki-xtc-gui-studio-trial/releases/latest

## 起動方法

1. `tategaki-xtc-gui-studio_v1.4.4.1-trial_windows-portable.zip` をダウンロードします。
2. zipファイルを展開します。
3. 展開したフォルダ内の `START_TATEGAKI_XTC_GUI.bat` を実行します。

Windows SmartScreen が表示された場合は、内容を確認したうえで「詳細情報」→「実行」を選んでください。
